# DPIA Template (Starter)

## Data Processed
- Pseudonymised patient reference (no NHS number/MRN by default)
- Medication orders
- Monitoring events
- Task workflow metadata

## Lawful Basis
- GDPR Article 9(2)(h): healthcare delivery

## Purpose
- Patient safety monitoring to prevent missed tests

## Data Sharing
- Internal Trust only
- Optional webhook integration (configurable)

## Retention
- Default 90 days (configurable)
- Longer retention requires formal IG decision
